Pingin'

React Native Firebase Geolocation

A real-time social location sharing platform designed to connect friends spontaneously.

The Problem

Social coordination is often friction-heavy. "Where are you?" texts and delayed responses make spontaneous meetups difficult. Existing map apps are either too broad (Google Maps) or too privacy-invasive (Snap Map).

The Solution

Pingin' focuses on ephemeral, consent-based location sharing. Users can send a "Ping" to specific friends, sharing their live location for a set duration, facilitating immediate meetups without constant surveillance.

Technical Highlights

  • Real-time Geolocation: Optimized background location tracking to balance battery life with accuracy.
  • Firebase Backend: Utilized Firestore for real-time data sync and Cloud Functions for notification logic.
  • Google Maps API: Custom map styling and marker clustering for a clean UI.
